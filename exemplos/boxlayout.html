O que são divs

Divs são a contração de divisões, separação, containers ou organizadores.

Em geral podemos fazer qualquer coisa com divs e css, é possível criar caixas organizadoras (containers), inserir planos de fundo, imitar elementos HTML entre outros mains.

Sua principal função no código é dar uma melhor organização flexível para o desenvolvedor e permitir que ele organize os elementos visuais de forma mais profissional.
